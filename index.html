Currently shipping

Livery

Local, single-user harness for running a small AI company.

Markdown and git — no database, no server, no cloud. Hire agents (Claude Code, Codex, Cursor, local Ollama or LM Studio models), file tickets, dispatch work, close the loop, all from the terminal. Multi-runtime by design: adding a new adapter is about thirty lines of Python. Pre-1.0, MIT-licensed.

Scribe

Native macOS dictation that runs entirely on your machine.

Press a global hotkey, speak, release — your words are transcribed locally with WhisperKit and pasted into whatever app is in front of you. No cloud, no account, no upload, no telemetry. macOS 14+, Apple Silicon recommended.
